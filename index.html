Asian College YouTube Channel RSS Fetcher

This website finds the YouTube RSS feed for a channel, from a channel URL. If the channel ID is in the URL, it just uses it, otherwise it checks the YouTube API to get the channel ID from the username.

For example:
https://www.youtube.com/user/vinesauce/ ➡️ https://www.youtube.com/feeds/videos.xml?channel_id=UCzORJV8l3FWY4cFO8ot-F2w
https://www.youtube.com/channel/UCYDnJiF0_RqSjkjvjRbG1tA ➡️ https://www.youtube.com/feeds/videos.xml?channel_id=UCYDnJiF0_RqSjkjvjRbG1tA
